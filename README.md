# Projeto Fundamento Python 2 Aula 2

## How to use this project

You first need to install virtual environemnt, by running the following command:

```bash
python3 -m venv .venv
```

After that, you have to activate the venv:

```bash
source .venv/bin/activate
```

Then, to install other packages:

```bash
pip install fastapi
pip install sqlalchemy
pip install regex
```
